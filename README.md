# Beef Check – Restaurant Website

This project is a restaurant website built using HTML and CSS.

The website includes several pages:
- Homepage
- Menu page
- Dessert page
- Contact & Reservation page

Features:
- Responsive layout
- Navigation between pages
- Contact and reservation form
- Restaurant menu presentation

Technologies used:
- HTML5
- CSS3
